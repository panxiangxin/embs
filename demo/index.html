结构化添加

用表单添加/编辑资产，会自动同步到 JSON 与列表。

自定义资产与标签

粘贴 JSON 或导入文件，应用后会清空索引，需要重新构建。

格式示例：[{ "id": "...", "name": "...", "labels": ["标签1","标签2"] }]
